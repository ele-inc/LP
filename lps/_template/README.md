# LP テンプレート使い方ガイド

## 概要

このテンプレートは、新しいランディングページ（LP）を素早く作成するための雛形です。
`ele-v2` のデザインをベースにしています。

**コードが書けなくても大丈夫！** AIに指示を出すだけでLPが作れます。

---

## プレビュー方法（HTMLを確認する）

### 方法1: VSCodeでプレビュー（おすすめ）

1. VSCodeで `index.html` を開く
2. 右クリック → 「Open with Live Server」を選択
3. ブラウザが自動で開き、リアルタイムでプレビューできます

※ Live Server拡張機能が必要です。インストールされていない場合:
- VSCode左側の拡張機能アイコンをクリック
- 「Live Server」で検索
- インストールボタンをクリック

### 方法2: ブラウザで直接開く

1. Finderで `index.html` を見つける
2. ダブルクリックするか、ブラウザにドラッグ&ドロップ

### 方法3: AIに頼む

Claude（このAI）に「プレビューサーバーを起動して」と伝えてください。

## AIを使ったLP作成フロー

### ステップ1: 新しいLPを作る

AIに伝える:
```
「○○」というサービスのLPを作りたい。
_templateをコピーして「○○-lp」フォルダを作って。
```

### ステップ2: 企画書を作る

AIに伝える:
```
brief.mdを埋めて。サービスの内容は以下:
- サービス名: ○○
- ターゲット: ○○
- 特徴: ○○
```

### ステップ3: コンテンツを作る

AIに伝える:
```
content.mdを埋めて。キャッチコピーは「○○」で。
```

### ステップ4: HTMLを完成させる

AIに伝える:
```
content.mdの内容をindex.htmlに反映して。
```

### ステップ5: 確認・修正

AIに伝える:
```
プレビューを起動して。
```

確認後:
```
ヒーローのキャッチコピーを「○○」に変えて。
ボタンの色をオレンジに変えて。
お客様の声を2つに減らして。
```

## クイックスタート

### 1. テンプレートをコピー

AIに伝える:
```
新しいLP「my-service」を作りたい。_templateをコピーして。
```

または手動で:
```bash
cp -r lps/_template lps/my-service
```

### 2. ファイルを編集

各ファイルの `[...]` で囲まれた部分を実際のコンテンツに置き換えます。
（AIに頼めば自動でやってくれます）

---

## ファイル構成

```
lps/_template/
├── README.md      # このファイル（使い方ガイド）
├── brief.md       # 企画書テンプレート
├── content.md     # コンテンツ原稿テンプレート
└── index.html     # LPコードテンプレート
```

---

## 各ファイルの役割

### brief.md（企画書）

LPの全体設計を記載します。最初にこのファイルを埋めることで、方向性を明確にできます。

**記載内容:**
- 基本情報（プロジェクト名、ステータス）
- サービス概要
- デザイン方針
- キャッチコピー
- セクション構成

### content.md（コンテンツ原稿）

LPに掲載する実際の文章を記載します。HTMLを編集する前に、このファイルでコンテンツを整理すると効率的です。

**記載内容:**
- ヒーローセクションのコピー
- お客様の声
- 悩み・課題
- 機能紹介
- サポート内容
- フォーム関連

### index.html（LPコード）

実際のLPページです。`content.md` の内容を元に `[...]` を置換してください。

---

## 作成フロー

```
1. brief.md を記入（企画・設計）
      ↓
2. content.md を記入（文章作成）
      ↓
3. index.html を編集（コーディング）
      ↓
4. プレビューで確認
      ↓
5. 修正・調整
```

## トラブルシューティング

### プレビューが表示されない

AIに伝える:
```
Live Serverでプレビューが開かない。別の方法で起動して。
```

### 変更が反映されない

- ブラウザをリロード（Cmd+R または Ctrl+R）
- キャッシュをクリア（Cmd+Shift+R または Ctrl+Shift+R）

### レイアウトが崩れた

AIに伝える:
```
レイアウトが崩れた。元に戻して。
```
