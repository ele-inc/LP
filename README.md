# LP

AI前提のLP構築の仕組みを構築する

## 構成

```
lps/           LPフォルダ（各LPはサブフォルダで管理）
template/      新規LP作成用テンプレート
README.md      このファイル
```

## LP一覧

| 名前   | パス        | 説明           |
| ------ | ----------- | -------------- |
| ele-v2 | lps/ele-v2/ | ZERO（メイン） |
| ele-v1 | lps/ele-v1/ | ELE（旧）      |

## 使い方

**編集**: 「ele-v2のタイトルを変えて」
**新規作成**: template/base.html をコピーして lps/新フォルダ/ に配置

## プレビュー

```
open lps/ele-v2/index.html
```
